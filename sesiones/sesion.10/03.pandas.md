### ðŸ”¹ Joins y merges (estilo SQL)

```python
clientes = pd.DataFrame({
    "id": [1, 2],
    "nombre": ["Ana", "Luis"]
})

ventas = pd.DataFrame({
    "id_cliente": [1, 1, 2],
    "producto": ["PC", "Monitor", "Teclado"],
    "precio": [800, 150, 20]
})

# INNER JOIN
df_merge = ventas.merge(clientes, left_on="id_cliente", right_on="id")
print(df_merge)
```

---

### ðŸ”¹ ConcatenaciÃ³n de DataFrames

```python

```
